A simple library to make staggering message rotation in a header.

The way it should work is:

  1. Display the first two lines right away.
  2. Animate the first line
  3. Animate the second line
  4. Repeat steps 2 and 3 until we reach the end of our list of text to show
  5. Reset both elements at the same time to show the first two lines.
  6. Repeat

Current bugs:

  • Weird white line above the animated text in Chrome. Not sure what's causing that.
  • When resetting text to the initial state only the top line animates. Not sure why the 2nd line isn't getting animated.
